# HH's ROOM: Found Myself

我在做一个个人网站，名称是 **HH's ROOM: Found Myself**。

核心气质：杂志感、温柔风、叙事感、Notion 风。

视觉细节：少量手写元素，例如 `Found Myself` 下方的 `still in progress`，`最近在研究什么` 下方的 `currently obsessed with...`，以及各模块旁边的英文小注释。

辅助色：`--accent-secondary: #3c506b`，用于 hover、link 和 timeline。

首页一句话：

> 真实、观察、迭代。

最近在研究：

> Cursor 如何帮助普通人做产品。

## 网站结构

- `index.html`：页面结构
- `styles.css`：视觉样式
- `script.js`：页面内容数据
- `CNAME`：自定义域名配置，计划使用 `hhs-room.com`

## 五个内容部分

1. 身体观察室：文字记录身体观察、运动反馈、睡眠变化、经期记录、恢复状态、水肿观察。
2. 人与人的连接：匿名案例文章，分析冲突、误解、沟通和关系样本。
3. 生活采样：短句记录、月度总结、舒服的东西、家具氛围、气味灵感、点亮我的瞬间。
4. Vibe Coding 日志：项目列表，记录我如何用 Cursor 把想法做成产品。
5. 成长轨迹：年份时间线，记录重大事件和每年在做什么。

模块手写注释：

- 身体观察室：`the body always knows`
- 人与人的连接：`between you and me`
- 生活采样：`collected moments`
- Vibe Coding 日志：`idea → prototype → repeat`
- 成长轨迹：`still becoming`
